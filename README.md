# Meta-Ad-analysis-PB
This is a real time data analysis
The business needs a performance tracking report for advertising campaigns running on Facebook and Instagram. This report will provide visibility into campaign reach, engagement, conversions, and budget
utilization.
This will enable the marketing team to:
• Identify the most effective platform (Facebook vs Instagram).
• Track campaign ROI and optimize budget allocation.
• Understand audience engagement patterns.

In this report you will see Campaigns running on Facebook and Instagram only.
